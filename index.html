S No. Name Opening Tag Closing Tag Attributes Description Syntax Example
1. DOCTYPE <!DOCTYPE> ~ Defines the document type <!DOCTYPE ....>
2. html <html> </html> Defines an HTML document <html>...</html>

11. img <img> No Closing Tag Defines an image <img src="link to image" alt="alternate text">

13. area <area> No Closing Tag Defines an area inside an image-map. The 'area' element is always nested inside a 'map' tag. <area shape="rect" coords="0,0,82,126" href="sun.htm" alt="Sun">
14. canvas <canvas> </canvas> Used to draw graphics, on the fly, via scripting (usually JavaScript).
The canvas tag is only a container for graphics, you must use a script to actually draw the graphics.		 <canvas id="mycanvas"></canvas>
