HTML Reference Sheet

S No. Name Opening Tag Closing Tag Attributes Description Syntax Example
1. Heading <h1> - <h6> </h1> - </h6> To add a [bold]heading (size decreases from h1 to h6) <h1>Heading</h1>
2. Paragraph <p> </p> To Add a Paragraph <p> Lorem ipsum Text </p>
3. Comment <!-- --> To improve readability of the code or to test code without deleting anything <!-- Here goes the comment -->
4. Image <img> ~ To add an image <img src="source-link" alt="alternative text">
5. Anchor <a> </a> To add a link to elements <a href="destination-link"> Click here to go to link </a>
6. Ordered List <ol> </ol> Create an Ordered List <ol><li>First</li> <li>Second</li></ol>
7. Unordered List <ul> </ul> Create an Unordered List <ul><li>First</li> <li>Second</li></ul>
8. Input <input> ~ To get input from user <input type="input-type">
9. Form <form> </form> To create a form to submit <form action="/url-to-submit-form-data"> all the input from user goes here </form>
10. Button <button> </button> To create a button <button type="button-type"> click me </button>
11. Division <div> </div> To group and characterise elements <div> some elements </div>
